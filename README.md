## Installation

```shell
git clone https://github.com/shopot/mobx-context-example.git

cd mobx-context-example

npm install
```

## Running the app

```shell
# development
$ npm run dev

# build production
$ npm run build

# preview the production build
$ npm run preview

# start linting
$ npm run lint
```
